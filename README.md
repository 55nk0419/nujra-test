# NUJRA test website

This is a GitHub Pages prototype for NUJRA.

## Files

- `index.html`: page content and structure
- `css/style.css`: design
- `js/main.js`: JavaScript for loading JSON data
- `data/members.json`: member data
- `data/events.json`: event/seminar data

## Editing

To update events, edit `data/events.json`.

To update members, edit `data/members.json`.

## Publish

Enable GitHub Pages from:

Settings > Pages > Deploy from a branch > main > /root
